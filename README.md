# commands

A collection of bash commands

## [changename](man/man1/changename.1.html)

Change the name of a Terminal window.

    changename [(-n|--name) <name>]
    changename [(-r|--reset)]
    changename [(-h|--help)]

## [colortext](man/man1/colortext.1.html)

Echo colored text.

    colortext <color> <text>
    colortext [-c|--color]
    colortext [-e|--examples]
    colortext [-h|--help]
